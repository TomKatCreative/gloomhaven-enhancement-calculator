# Gloomhaven Enhancement Calculator & Character Sheets

A mobile companion app for the Gloomhaven board game series. Track your characters and calculate enhancement costs with edition-specific rules.

Available on [Google Play](https://play.google.com/store/apps/details?id=tomkatcreative.gloomhavenenhancementcalc) and the [App Store](https://apps.apple.com/app/id1586920865).

## Features

### Enhancement Calculator
- Calculate enhancement costs with edition-specific rules
- Supports all enhancement types: +1 stats, elements, conditions, summon stats, and more
- Accounts for card level, previous enhancements, and multi-target abilities
- Edition-specific modifiers (Lost, Persistent, Temporary Enhancement)
- Frosthaven Building 44 enhancer upgrades
- Detailed cost breakdown showing how the final price is calculated

### Character Sheets
- Create and manage multiple characters across all supported game editions
- Track gold, XP, checkmarks, and resources
- Perk tracking with attack modifier deck visualization
- Mastery tracking for Frosthaven characters
- Retirement tracking with previous retirements affecting new character bonuses
- Support for locked/unlocked classes with spoiler protection

### Supported Game Editions
- **Gloomhaven** (1st Edition)
- **Gloomhaven** (2nd Edition)
- **Jaws of the Lion**
- **Frosthaven**
- **Crimson Scales** (fan expansion)
- **Mercenary Packs** (standalone character packs)
- **Custom Classes** (community-created content)

### Quality of Life
- Dark mode support
- Multiple theme color options
- Element tracker for managing infusions during gameplay
- Clean, intuitive interface optimized for mobile
- Available in English and Portuguese

## Built With

Written in Dart using the [Flutter](https://flutter.dev) framework.
